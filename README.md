# Mt. Diablo Iris Society Website

This is the official website for the Mt. Diablo Iris Society, a group dedicated to promoting interest in iris flowers in the Mt. Diablo area.

## Structure

- `index.html`: Home page
- `pages/`: Additional pages (About Us, Events, Gallery, About Iris, Resources)
- `css/styles.css`: Stylesheet
- `js/scripts.js`: JavaScript file
- `images/`: Directory for images

## How to Run Locally

To view the website locally, you can use a simple HTTP server.

### Using Python (if available)

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

### Using Node.js (if available)

Install `http-server` globally:

```bash
npm install -g http-server
```

Then run:

```bash
http-server
```

Open `http://localhost:8080` in your browser.

## Deployment

This is a static website that can be deployed to any static hosting service like GitHub Pages, Netlify, or Vercel.

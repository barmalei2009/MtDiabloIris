# Event Submission Template

Use this template to add new events to the events.json file. Copy and fill out the fields for each new event.

---

- **name**: Event name (e.g., Monthly Meeting)
- **date**: Event date (e.g., September 13, 2026)
- **place**: Event location (e.g., Pleasant Hill 320 Civic Drive Pleasant Hill, CA 94523)
- **time**: Event time (optional, e.g., 1:00 to 3:00 PM)
- **note**: Additional notes (optional)

---

## Example

```
{
  "name": "Monthly Meeting",
  "date": "September 13, 2026",
  "place": "Pleasant Hill 320 Civic Drive Pleasant Hill, CA 94523",
  "time": "1:00 to 3:00 PM",
  "note": "Guest speaker: Jane Doe"
}
```

Add your new event object to the events.json array, separated by commas.

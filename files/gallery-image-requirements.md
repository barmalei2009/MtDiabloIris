# Gallery Image Submission Requirements

To ensure a high-quality and consistent gallery, please follow these requirements when submitting images:

## Image Size
- **Minimum Dimensions:** 1200 x 800 pixels (width x height)
- **Maximum Dimensions:** 4000 x 3000 pixels
- **Aspect Ratio:** Preferably 3:2 or 4:3, but other ratios are accepted if the subject is well-framed

## File Format
- Accepted formats: JPEG (.jpg, .jpeg), PNG (.png)
- No HEIC, TIFF, or RAW files

## File Size
- **Maximum file size:** 4 MB per image
- **Minimum file size:** 200 KB (to ensure quality)

## Image Quality
- Images must be in focus, well-lit, and free of heavy compression artifacts
- No watermarks, borders, or text overlays
- No excessive filters or color manipulation
- Subject (iris flower or event) should be clearly visible and centered

## File Naming
- Use descriptive filenames, e.g., `TB_SpringBeauty_2026.jpg`
- Avoid spaces and special characters

## Metadata (Optional but encouraged)
- Include iris class, variety name, and photographer in the submission form or email

---

**Submissions not meeting these requirements may be rejected or edited for consistency.**

For questions, contact the gallery administrator.
